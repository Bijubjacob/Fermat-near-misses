# Fermat Near Miss Finder

## Description
This program searches for "near misses" to Fermat’s Last Theorem.  
It checks values of **x** and **y** (from 10 to k) and finds a **z** such that:

x^n + y^n ≈ z^n

The program prints a message every time it finds a new smallest relative miss.

## How to Run
1. Open the file in Python
2. Run the program
3. Enter:
   - **n** (must be between 3 and 11)
   - **k** (must be 10 or higher)

## Output
The program prints:
- x, y, n
- z
- miss (integer difference)
- relative miss (percentage)
